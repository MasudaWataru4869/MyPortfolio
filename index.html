Vision

絵本は人を惹きつける

人を惹きつけれる
デザイナーに

絵本のようなデザインを目指しています。

絵本は、誰が読んでもわかりやすいように作られてます。

それでいて何度読んでもおもしろい。アートのような絵やパッとひと目見て読みたいと思わせるようなインパクト。

子供たちがページをめくるたびに興奮し、惹きつけるられる工夫。様々な考えや思惑が存在していると考えます。

絵本のような伝わりやすく読みやすい、ページをめくるたびに興奮し、見た人を惹きつけるようデザインを目指しています。

Skills

HTML

下層ページも含めWEBサイトを1から制作することが可能です。基本的なコーディングは問題ありません。メインエディタは、AWS Cloud9を使っています。

CSS

レスポンシブやsass・アニメーションを含め、基本的なところは問題なく、実装できます。また、不具合があってもディベロッパーツールを使い解決できます。

Javascript

複雑なコードなどは書くことができませんが、プラグインの導入やクラス名の付与などの簡単なコードは実装できます。まだまだ調べながら実装することがほとんどです。

Photoshop

Adobeで配信されている無料動画やYoutubeを参考に操作を学習しました。主にデザインカンプの制作と写真の編集に使用しています。デザインカンプの制作や写真の補正・加工・切り抜きといった画像の編集など作業が可能です。わからないことやできないことでも調べれば操作・実装ができます。

Illustrator

Adobeで配信されている無料動画やYoutubeを参考に操作を学習しました。主にワイヤーフレームの制作やイラスト作りに使用しています。簡単なロゴ・アイコンの作成などの作業が可能です。ただ複雑なロゴや複雑なアイコンなどの制作は、時間がかかってしまいます。

Lightroom

大学生の頃からカメラが趣味で約6年ほど使用しています。基本的な操作は、ひと通りできます。

Steering

船を操船できます。沖縄の石垣島に住んでる時は、仕事で毎日、船を操船していました。車の運転よりも慣れております。経験上最大約50フィート(約15m)までは操船可能です。

Teaching

大学時代は、教育やスポーツ指導について学び教員免許を所持しています。またダイビングのインストラクター経験もあります。人に物事を教えることは、苦ではないです。むしろ好きです。
